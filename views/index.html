API Project: URL Shortener Microservice

User Story:

  1. If you POST a valid URL to [project_url]/api/shorturl/new, you will receive a shortened URL in the JSON response.
    Example : {"original_url":"www.google.com","short_url":1}
  2. If you pass an invalid URL that doesn't follow the http(s)://www.example.com(/more/routes) format, the JSON response will contain an error like {"error":"invalid URL"}
  3. If you visit the shortened URL, it will redirect you to your original link.

Short URL Creation

Example Usage:

[this_project_url]/api/shorturl/3
